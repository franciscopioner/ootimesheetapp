# Registro de horários - Ponto virtual

## Instalação:

- **npm install** ou **yard install**
- **npm start** ou **yard start**



## Instruções:

A aplicação se baseia em um empresa com as seguintes rotinas:

​	**Jornada:** 09hs - 12hs - 13hs - 18hs
​	**Máximo:** 10hs/dia

A aplicação suporta registro do horário de entrada a partir das 08hs da manhã.

O funcionário não pode exceder 2hs extras diárias.
